headinds and paragraphs

Heading tags in HTML are used to define titles and subtitles on a webpage. They range from h1 to h6

  • h1 represnt the most important (main heading)
  • h6 represents the least (smalest heading)

Paragraphs

The paragraph tag "P" in HTML is used to define a block
of text as a paragraph. It automatically adds space
before and after the text to make content more readable.

lists

A list is a way to group related items together on a
webpage. HTML provides different types of lists to
organize content

order list

An ordered list is a list of items that are numbered
automatically (1, 2, 3 …).It is created using the "ol"

unorder list

An unordered list is a list of items with bullet points
(•).It is created using the :ul"

Examples

order list

  1. HTML5
  2. CSS3
  3. javascript
  4. git

unorder list
